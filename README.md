# Decision-Tree-Optimization-using-Pre-Pruning-and-Post-Pruning
This project demonstrates the implementation and optimization of Decision Tree classifiers using Machine Learning techniques in Python.

The notebook covers:

* Data preprocessing
* Exploratory Data Analysis (EDA)
* Decision Tree modeling
* Overfitting analysis
* Pre-Pruning techniques
* Post-Pruning using Cost Complexity Pruning (CCP Alpha)
* Feature importance analysis
* Performance comparison across multiple models

Key Concepts Explored

* Decision Trees
* Gini Impurity
* Overfitting vs Generalization
* Max Depth
* Minimum Samples Split
* CCP Alpha Pruning
* Model Evaluation Metrics

Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

Model Evaluation

The project compares:

1. Default Scikit-learn Decision Tree
2. Pre-Pruned Decision Tree
3. Post-Pruned Decision Tree

using:

* Accuracy
* Precision
* Recall
* F1 Score

The analysis shows how pruning helps simplify tree complexity while improving model generalization.
